#  Custom Event Calendar

A responsive, interactive event calendar built using **React**, **Tailwind CSS**, and **React Beautiful DnD**. This application allows users to create, manage, and organize their events seamlessly through an intuitive drag-and-drop interface.

---

##  Features

-  **Month View Calendar** — Visual layout of all events by day.
-  **Global State Management** — Handled with Context API.
-  **Create, Edit & Delete Events** — Easily manage event data.
-  **Recurring Events** — Customizable patterns for daily/weekly/monthly recurrence.
-  **Drag & Drop** — Reorder events with React Beautiful DnD.
-  **Persistent Storage** — Automatically saves to localStorage.
-  **Search & Filter** — Quickly find events.
-  **Tailwind CSS** — Clean, responsive design.
-  **Date-fns** — Lightweight and powerful date utilities.

**Install dependencies:**
Ensure you are in the correct directory. 
```bash
npm install
```

**Start the development server:**
```bash
npm run dev
```

Open your browser and navigate to:  
[http://localhost:5173](http://localhost:5173)

## Main Libraries Used

- React – Core library for building the UI.

- Tailwind CSS – For responsive and modern styling.

- react-beautiful-dnd – For drag-and-drop support.

- Vite – For fast development and optimized builds.
